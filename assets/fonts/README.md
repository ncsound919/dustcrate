# Velum Stroke Font

Place `VelumStroke.ttf` in this directory before building.

Velum Stroke is a handwritten graffiti tag typeface by Cikareotype Studio.
- Personal use demo: https://www.fontspace.com/velum-stroke-font-f157114
- Commercial license: https://cikareotype.com/product/velum-stroke-natural-graffiti-tags-font/

Purchase a license and place the TTF file here as `VelumStroke.ttf`.
CMakeLists.txt will compile it into the plugin binary via `juce_add_binary_data`.
