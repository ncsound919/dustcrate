# Velum Stroke Font Implementation — Final Steps

## Overview
This document provides the **exact code changes** needed to complete the Velum Stroke font integration for the DustCrate synth header. All preliminary work (CMakeLists.txt, PluginEditor.h, assets/fonts/ directory) has been committed. Only `PluginEditor.cpp` remains.

---

## Required Changes to `Source/PluginEditor.cpp`

### 1. Load Typeface in Constructor

Add the following code **at the end** of the `DustCrateAudioProcessorEditor` constructor (after `refreshBrowsers();`):

```cpp
// Load Velum Stroke typeface from compiled binary data.
// BinaryData::VelumStroke_ttf is generated by juce_add_binary_data.
#if JUCE_MODULE_AVAILABLE_juce_core
  if (BinaryData::namedResourceListSize > 0)
  {
    // Iterate BinaryData to see if VelumStroke_ttf exists
    bool foundVelum = false;
    for (int i = 0; i < BinaryData::namedResourceListSize; ++i)
    {
      if (juce::String(BinaryData::namedResourceList[i]).contains("VelumStroke"))
      {
        foundVelum = true;
        break;
      }
    }
    
    if (foundVelum)
    {
      // Load the font from binary data
      velumStrokeTypeface = juce::Typeface::createSystemTypefaceFor(
        BinaryData::VelumStroke_ttf,
        BinaryData::VelumStroke_ttfSize
      );
    }
  }
#endif

// Fallback is handled in paint() — if velumStrokeTypeface is null, Courier New is used.
```

---

### 2. Use Velum Stroke in `paint()` Method

Replace the existing font line in the `paint` function.

**OLD CODE (line ~465):**
```cpp
g.setFont(juce::Font("Courier New",20,juce::Font::bold));
```

**NEW CODE:**
```cpp
// Use Velum Stroke if loaded, otherwise fall back to Courier New
if (velumStrokeTypeface != nullptr)
  g.setFont(juce::Font(velumStrokeTypeface).withHeight(24.0f)); // Velum Stroke is expressive, slightly larger
else
  g.setFont(juce::Font("Courier New",20,juce::Font::bold)); // Fallback
```

## Troubleshooting

### Font not loading?

1. Verify `assets/fonts/VelumStroke.ttf` exists
2. Check CMake output for "VelumStroke_ttf" in the BinaryData target
3. Confirm BinaryData is linked: `target_link_libraries(DustCrate PRIVATE DustCrateAssets)`
4. Rebuild from scratch (`rm -rf build/ && mkdir build && cd build && cmake .. && cmake --build .`)

### Font looks too small/large?

Adjust the font height in `paint()`:
```cpp
g.setFont(juce::Font(velumStrokeTypeface).withHeight(22.0f)); // Try different sizes
```
