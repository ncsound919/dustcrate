# Bad Mofo Font Implementation - Complete Guide

## Overview
This guide provides **complete, ready-to-use code** for integrating the **Bad Mofo Regular** graffiti font into the DustCrate synth header. Bad Mofo is **free for commercial use** — no license purchase required!

## 📝 Remaining Steps

### 1. Download Bad Mofo Font

Download `BadMofo.ttf` from:
- https://www.1001fonts.com/bad-mofo-font.html
- https://www.dafont.com/bad-mofo.font  
- https://www.fontspace.com/bad-mofo-font-f5276

Place the file at: `assets/fonts/BadMofo.ttf`

### 2. Update `Source/PluginEditor.h`

Replace the `velumStrokeTypeface` declaration with `badMofoTypeface`:

**Find (line ~122):**
```cpp
juce::Typeface::Ptr velumStrokeTypeface;
```

**Replace with:**
```cpp
juce::Typeface::Ptr badMofoTypeface;  // Bad Mofo graffiti font for hip-hop header
```

### 3. Update `Source/PluginEditor.cpp` — Constructor

Add the following **at the end** of the `DustCrateAudioProcessorEditor` constructor (after `refreshBrowsers();`):

```cpp
// Load Bad Mofo typeface from binary data.
// BinaryData::BadMofo_ttf is generated by CMake's juce_add_binary_data.
#if JUCE_MODULE_AVAILABLE_juce_core
  // Check if BadMofo.ttf was compiled into BinaryData
  try {
    badMofoTypeface = juce::Typeface::createSystemTypefaceFor(
      BinaryData::BadMofo_ttf,
      BinaryData::BadMofo_ttfSize
    );
  }
  catch (...) {
    // If font not found, badMofoTypeface stays nullptr → fallback to Courier New
    DBG("Bad Mofo font not found in BinaryData, using fallback font.");
  }
#endif
```

### 4. Update `Source/PluginEditor.cpp` — paint() Method

**Find (line ~465):**
```cpp
g.setFont(juce::Font("Courier New",20,juce::Font::bold));
```

**Replace with:**
```cpp
// Use Bad Mofo if loaded, otherwise fall back to Courier New
if (badMofoTypeface != nullptr)
  g.setFont(juce::Font(badMofoTypeface).withHeight(26.0f));  // Bad Mofo is bold & expressive
else
  g.setFont(juce::Font("Courier New",20,juce::Font::bold));  // Fallback
```

## Troubleshooting

### Font not loading?

1. Verify `assets/fonts/BadMofo.ttf` exists
2. Check CMake output for `BadMofo_ttf` in BinaryData
3. Rebuild from scratch:
   ```bash
   rm -rf build/
   mkdir build && cd build
   cmake .. && cmake --build .
   ```

### Font looks wrong?

Adjust the height in `paint()`:
```cpp
g.setFont(juce::Font(badMofoTypeface).withHeight(24.0f));  // Try different sizes
```
